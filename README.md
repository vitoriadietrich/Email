# Email Subscription Box Using HTML & CSS | Newsletter Section

Um projeto simples de **caixa de inscrição de email (newsletter)** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar uma **newsletter section moderna**, ideal para capturar emails de usuários em sites.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Campo de email estilizado
- Botão de inscrição
- Design simples e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
